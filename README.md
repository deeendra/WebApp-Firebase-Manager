# Firestore Manager

Enterprise-grade web application to manage Firestore collections, detect schema anomalies (ghost fields), and perform CRUD operations directly on documents.

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Shadcn UI (Custom Dark/Bumblebee Theme)
- Firebase Client SDK

## Features
- **Authentication**: Restricted access via Google Sign-in to specific admin emails.
- **Dynamic Schema Detection**: Automatically scans collections to extract schemas and detect inconsistent "ghost fields" generated by AI.
- **Data Explorer**: Excel-like table view to view, edit, add, and delete documents dynamically.
- **Schema Management**: Rename or delete specific fields across all documents in a collection safely.

## Setup Instructions

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.local` variables from your Firebase console:
   ```bash
   cp .env.local.example .env.local
   ```
   Fill in the required Firebase credentials.
4. Run the development server:
   ```bash
   npm run dev
   ```

## Deployment
This project is configured to be deployed on **Vercel**. Link your GitHub repository to Vercel and it will automatically deploy the `main` branch.
